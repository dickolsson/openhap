# spec/

Applies when working on files under `spec/`.

## Purpose

`spec/*.md` are generated protocol references, extracted from the reference
sources in the gitignored `external/` directory (populated by the
`fetch-external` skill):

- `HAP.md` — index and overview; entry point into the `HAP-*.md` topic files
  (generated by the `spec-hap` skill)
- `HAP-*.md` — HAP topic files: TLV8, pairing, encryption, HTTP, mDNS, services,
  characteristics, categories (generated by the `spec-hap` skill)
- `MQTT.md` — Tasmota MQTT protocol (generated by the `spec-mqtt` skill)
- `IMPLEMENTATIONS.md` — practical patterns from the reference implementations
  (generated by the `implementations` skill)

## Writing changes

Do not hand-edit generated files beyond typo fixes — regenerate them with the
skill that owns them. If a spec file is missing or stale, run the owning skill
(after `fetch-external` if `external/` is absent).
