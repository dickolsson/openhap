=head1 NAME

OpenHAP::Pairing - HAP pairing protocol implementation

=head1 SYNOPSIS

    use OpenHAP::Pairing;

    my $pairing = OpenHAP::Pairing->new(
        pin => '123-45-678',
        storage => $storage,
        accessory_ltsk => $ltsk,
        accessory_ltpk => $ltpk,
    );

    my $response = $pairing->handle_pair_setup($body, $session);
    my $response = $pairing->handle_pair_verify($body, $session);

=head1 DESCRIPTION

This module implements the HAP pairing protocol including pair-setup
(SRP) and pair-verify (X25519) flows.

=cut
