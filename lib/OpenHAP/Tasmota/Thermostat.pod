=head1 NAME

OpenHAP::Tasmota::Thermostat - Tasmota thermostat HAP accessory

=head1 SYNOPSIS

    use OpenHAP::Tasmota::Thermostat;

    my $thermostat = OpenHAP::Tasmota::Thermostat->new(
        aid => 2,
        name => 'Bedroom Thermostat',
        mqtt_topic => 'tasmota_AABBCC',
        mqtt_client => $mqtt,
    );

=head1 DESCRIPTION

This module implements a HAP thermostat accessory for Tasmota devices
with temperature sensor and relay control.

=cut
